# 의사결정 체크리스트 — 실거주 vs 투자

집을 고를 때 두 가치를 분리해서 평가한다. 하나의 점수로 뭉개면 판단이 흐려진다.

## A. 실거주 가치 (지금 우리가 살기 좋은가)

- [ ] 출퇴근: 두 사람 직장까지 문 to 문 시간. 환승 횟수.
- [ ] 생활 인프라: 마트/병원/공원. 아이 계획 있으면 학군·어린이집.
- [ ] 신축도: 준공연도. 2010년 이후면 주차·단열·구조가 크게 개선됨.
- [ ] 평형/구조: 신혼 2인 → 향후 확장 고려. 59㎡(24평) vs 84㎡(34평).
- [ ] 층/향: 저층·북향은 실거래가에서 이미 할인됨. 실거주 만족도는 별개.

## B. 투자 가치 (나중에 오를 여지가 있는가)

- [ ] 재개발/재건축 기대: 정비구역 지정 여부, 진행 단계, 대지지분.
- [ ] 가격 선반영: 기대감이 이미 가격에 얼마나 붙었나 (덜 붙을수록 업사이드).
- [ ] 대단지 여부: 세대수 많을수록 환금성·관리 유리.
- [ ] 희소성/입지: 역세권, 브랜드, 한강/공원 조망.
- [ ] 공급 물량: 인근 신축 입주 폭탄이 예정돼 있으면 단기 하방 압력.

## C. 자금 계획 (월 부담으로 환산)

- [ ] 매매: LTV 한도 × 주택담보대출 금리 → 월 원리금.
- [ ] 전세: 전세자금대출 금리 × 보증금 → 월 이자. 매매 월부담과 비교.
- [ ] 총 예산 = 매매가/보증금 + 취득세·중개비·이사비 + 대출 부대비용.
- [ ] 금리는 공시 최저가 아니라 "내 신용도에서 실제 나오는 금리"로 재계산.
- [ ] 상환 여력: 월 소득 대비 원리금 비중 (DSR 40% 선 참고).

## 종합 판단

- 실거주 비중이 크면 → `--lens live`, 출퇴근·생활 인프라 우선.
- 몇 년 뒤 매도/상급지 이동 계획이면 → `--lens invest`, 재개발·가격여력 우선.
- 신혼은 보통 "당장 살기 + 3~5년 뒤 자산" 둘 다 필요 → 두 렌즈 모두 돌려 교집합을 찾는다.

## 원본 워크플로우 메모

이 툴킷은 실제 신혼집 탐색(2026년) 흐름을 재현한 것:
1. 출퇴근 기준으로 노선(7호선 등) 주변 예산대 매물 스캔
2. 출퇴근 빼고 순수 투자가치로 지역 재순위 (한강 이남 등)
3. 1순위 지역 확정 후 평형별 실거래 후보 추출
4. 실거주형/투자형으로 후보를 나눠 최종 비교
